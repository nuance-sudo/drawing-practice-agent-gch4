# 審査中ステート表示の実装

## 課題
- GitHub Issue: [#54 審査開始後にステートをリセットする](https://github.com/nuance-sudo/drawing-practice-agent-gch4/issues/54)

## 要求
「この画像を審査する」ボタンを押した後、審査中であることをユーザーに明確に伝えるUIを表示する。

## 受け入れ条件
1. 「この画像を審査する」ボタンを押すと、ボタンがローディング状態に変化する
2. 審査中はボタンに「審査中...」テキストとスピナーが表示される
3. 審査中は画像のクリアボタン（×）が無効化される
4. 審査完了後はアップロードエリアがリセットされる（既存動作）
