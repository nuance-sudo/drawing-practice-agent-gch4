# Issue #52: ランク表示の修正とユーザープロフィール表示追加

## 要求概要

GitHub Issue: https://github.com/nuance-sudo/drawing-practice-agent-gch4/issues/52

### 背景

現在の審査結果ページに表示されるランクは、ユーザーの「現在のランク」を表示しているが、
本来は「審査実行時点のランク」を表示すべきである。

また、ユーザーの現在のランクを確認する機能がないため、
ヘッダーにプロフィール表示機能を追加し、そこから現在のランクを確認できるようにする。

### 変更内容

1. **審査実行時のランクを表示**
   - 審査タスクに実行時のランクを保存
   - 結果画面で保存されたランクを表示

2. **ユーザープロフィール表示機能の追加**
   - ヘッダー右上にGitHubプロフィール画像を表示
   - クリックでドロップダウンメニュー（現在のランク表示、ログアウトボタン）

### ユーザーストーリー

**US1**: ユーザーとして、審査結果を確認する際に、審査実行時点のランクを確認したい。
理由：スコアがそのランク相当かどうかを正しく判断するため。

**US2**: ユーザーとして、ヘッダーから自分の現在のランクをいつでも確認したい。
理由：上達の進捗を把握するため。

### 受け入れ条件

1. 審査結果画面で「審査時のランク」が正しく表示される
2. 新規審査では、審査実行時点のランクがタスクに保存される
3. ヘッダー右上にGitHubプロフィール画像が表示される
4. プロフィールをクリックすると現在のランクが表示される
5. ログアウトボタンが機能する

### 制約事項

- 既存のテストはすべてパスすること
- 既存のタスクデータへの後方互換性を維持（rank_at_reviewがない場合はフォールバック）
